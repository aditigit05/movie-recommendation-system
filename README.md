# 🎬 Movie Recommendation System

A basic content-based Movie Recommendation System built using Python and Machine Learning concepts.  
The system recommends similar movies based on genre similarity using TF-IDF Vectorization and Cosine Similarity.

---

## 🚀 Features

- Recommend movies based on genre similarity
- Uses Machine Learning similarity algorithms
- Simple and beginner-friendly implementation
- Built completely from scratch
- Console-based recommendation system

---

## 🛠️ Tech Stack

- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorization
- Cosine Similarity

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/aditigit05/movie-recommendation-system.git
```

Move into the project folder:

```bash
cd movie-recommendation-system
```

Install required libraries:

```bash
pip install pandas scikit-learn
```

---

## ▶️ Run the Project

```bash
python movie_recommendation.py
```
